# PR-Final-Project
Project Summary
A comprehensive Excel-based business analytics solution analyzing 250 customer transactions to generate insights, KPIs, and interactive visualizations.
What's Included
10 Professional Worksheets:

Raw transaction data (250 records)
Advanced formula analysis with DATE/TIME functions
Customer & product performance rankings
Regional sales breakdown
What-If scenario planning
Interactive charts (Bar, Pie)
Executive dashboard with live KPIs
Business insights report

Key Features
✅ 140+ Working Formulas - SUMIF, INDEX/MATCH, FILTER, DATEDIF, EOMONTH, IF statements
✅ Visual Analytics - Professional charts and conditional formatting
✅ Interactive Dashboard - Real-time KPIs with top customers and products
✅ What-If Analysis - Scenario planning with adjustable variables
✅ Zero Errors - All formulas verified and functional
Business Metrics

Total Revenue: $375,000+

50 Unique customers (Premium/Standard/Basic segments)
10 Products in 3 categories

Skills Demonstrated
Advanced Excel functions, data analysis, business intelligence, dashboard design, formula construction, data visualization, and professional reporting.
